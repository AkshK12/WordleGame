# WordleGame
Implementation of popular Wordle game.

Objective: Guess the secret five-letter word in six attempts.  
First Guess: Type any valid five-letter word and press Enter.  
Feedback Colors:  
Green: The letter is correct and in the right spot.  
Yellow: The letter is in the word but in the wrong spot.  
Red: The letter is not in the word at all.  
Subsequent Guesses: Use the color clues to refine your next guess, aiming to use letters that were yellow or green in new positions.  
Valid Words: All guesses must be real English words, and proper nouns, plurals, or offensive words are generally excluded from being answers.
